OverlayItem

OverlayItem class Maps & 3D Scene

Class representing an overlay item.

This class should not be instantiated directly.

An item shown on the map as a single element. Contains information such as name, image and coordinates. Common operations are reading identifiers and category/context info, obtaining coordinates and images and the display label, checking whether the item belongs to a overlay type, requesting extended preview data.
