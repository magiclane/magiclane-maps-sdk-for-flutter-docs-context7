progressMultiplier property

int get progressMultiplier

Get the progress multiplier.

SDK uses floating point values in the 0, 1 closed interval for the progress on operations. The values received in the notifyProgress notification are integers obtained by multiplying the internal floating point value with the value returned by progressMultiplier. By default, this function returns 100 so that the progress values that may be received will be between 0 and 100.

Returns

  • The progress multiplier
