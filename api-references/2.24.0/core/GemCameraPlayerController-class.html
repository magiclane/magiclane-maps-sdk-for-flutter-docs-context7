GemCameraPlayerController

GemCameraPlayerController class Sensor Data Source

A controller that manages playback of a camera feed from a DataSource.

Wraps a ValueNotifier and tracks camera data as GemCameraPlayerValue. Use with GemCameraPlayer to display frames from a camera source.
