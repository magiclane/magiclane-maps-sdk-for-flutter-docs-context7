improvedPosPreferRouteSnap constant

String const improvedPosPreferRouteSnap

Whether to snap to the route (instead of the most probable link). If true, the arrow will always snap to the route (if the route exists, and the distance is within the "snapToMapLinkThreshold"). If false, it will snap to the most probable map link, which is not necessarily the route link. This is only for display; does not affect the matching and guidance algorithm.
