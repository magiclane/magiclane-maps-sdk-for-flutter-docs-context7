DataSource class Sensor Data Source

Provides access to sensor and log data from various sources.

A DataSource represents a provider of sensing data â€” this can be a live sensor feed, a log file (playback), a simulated route, or an external source. It exposes methods to start/stop the source, query available data types, push mock or custom data, and register listeners to receive updates.

Use the dataSourceType property to determine whether the instance supports playback operations and access playback when applicable.
