AddressField

AddressField enum Landmarks

Address fields available on an AddressInfo.

Each enum value identifies a specific component of a postal address (for example city, streetNumber, postalCode). Use these values with AddressInfo.getField and AddressInfo.setField to read or modify individual address components.

See also:

  • AddressInfo - Structured address information for a landmark.
