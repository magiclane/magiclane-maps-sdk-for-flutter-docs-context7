OffBoardListener

OffBoardListener class Settings

Listener receiving offboard/content-related events from the SDK.

OffBoardListener delivers notifications about connectivity, content updates, resource deployment and auto-update progress. Register callbacks using the registerOn* methods to receive these events. Use SdkSettings.offBoardListener to obtain the singleton instance used by the SDK.

See also:

  • ContentStore — APIs that trigger many of the content-related callbacks delivered by this listener.
  • SdkSettings.offBoardListener — Obtain the global offboard listener instance.
