TurnEvent

TurnEvent enum Common

Navigation turn events.

Information about the manoeuvre that can be used by UI.

The enum contains common manoeuvres (e.g. straight, right, left), roundabout-related events, exit numbers and waypoint markers. Values are stable and intended to be consumed directly by client code.
