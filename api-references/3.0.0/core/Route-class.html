Route class

This class should not be instantiated directly. Instead, use the RoutingService.calculateRoute to retrieve a list of routes.

A concrete representation of a computed route that combines rich metadata, geometry and child objects for rendering, navigation and analysis. It models a full route result.

Common operations include operations available from RouteBase and converting the instance into a more specific route subtype when that category applies, and reading or updating auxiliary metadata attached to the route.

Can be presented on the map using the MapViewRoutesCollection associated with a GemMapController. Can be used for navigation using the NavigationService.
