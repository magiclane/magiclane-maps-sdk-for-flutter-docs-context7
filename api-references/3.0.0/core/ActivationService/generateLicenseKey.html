Generates a new license key for the given application and productId.

Parameters

  • IN applicationId The application id for which the license key is created.
  • IN productId The kind of product that the key will be bound to.
  • IN onComplete Will be invoked when the activation operation is completed, providing the error code.
    • GemError.success and the licenceKey if the activation was successfully completed.
    • GemError.required in case there's no internet connection or activation service could not be reached.
    • GemError.invalidInput if the input is invalid. Check hint for details.
    • GemError.noMemory in the rare ocasion when allocations are not possible.
    • GemError.networkFailed in case there was an internet connection but the generation request failed to start.

Returns

  • Associated TaskHandler for this operation if the search can be started otherwise null.

Throws

  • An exception if it fails.

Implementation

static TaskHandler? generateLicenseKey({
  required final String applicationId,
  final String productId = ProductID.core,
  required final void Function(GemError error, String hint) onComplete,
}) {
  final EventDrivenProgressListener progListener =
      EventDrivenProgressListener();
  GemKitPlatform.instance.registerEventHandler(progListener.id, progListener);

  progListener.registerOnCompleteWithDataCallback((
    final int err,
    final String hint,
    final Map<dynamic, dynamic> json,
  ) {
    GemKitPlatform.instance.unregisterEventHandler(progListener.id);
    onComplete(GemErrorExtension.fromCode(err), hint);
  });

  final String resultString = GemKitPlatform.instance.callObjectMethod(
    jsonEncode(<String, Object>{
      'id': 0,
      'class': 'ActivationService',
      'method': 'generateLicenseKey',
      'args': <String, dynamic>{
        'applicationId': applicationId,
        'productId': productId,
        'listener': progListener.id,
      },
    }),
  );

  final dynamic decodedVal = jsonDecode(resultString);
  final GemError errorCode = GemErrorExtension.fromCode(decodedVal['result']);

  if (errorCode != GemError.success) {
    return null;
  }

  return TaskHandlerImpl(progListener.id);
}
