GemCameraPlayer

GemCameraPlayer class Sensor Data Source

A widget that renders a camera feed from a GemCameraPlayerController.

Handles native image rendering, iOS rotation quirks, and flexible layout.

Available only for cameras where Camera.rgba8888 is available. May not available for all Camera provided by external data sources.
