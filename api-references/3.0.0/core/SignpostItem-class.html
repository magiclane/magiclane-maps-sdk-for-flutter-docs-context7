SignpostItem

SignpostItem class Core

SignpostItem object.

This class should not be instantiated directly. Instead, use the SignpostDetails.items getter to obtain a list of instances.

A single element of a rendered navigation signpost, used to assemble on‑screen sign displays and accessibility. Main fields carry the element’s semantic type and display content (text/phoneme), pictogram or shield classification, layout hints and ambiguity/connection flags; common uses are selecting icons/styles and positioning elements.
