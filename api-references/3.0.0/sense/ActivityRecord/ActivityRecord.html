Creates an ActivityRecord with the specified properties.

Parameters

  • IN shortDescription A short description of the activity. Default is empty string.
  • IN longDescription A detailed description of the activity. Default is empty string.
  • IN sportType The type of sport for this activity. Default is SportType.unknown.
  • IN effortType The effort level for this activity. Default is EffortType.easy.
  • IN bikeProfile The bicycle profile, if the activity involves cycling. Default is null.
  • IN visibility The visibility settings for the activity. Default is ActivityVisibility.everyone.
