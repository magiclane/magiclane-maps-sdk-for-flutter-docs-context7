GemError

GemError enum Core

Error codes.

Many methods in the SDK return a GemError value to indicate the success or failure of the operation.

The GemError.success and GemError.scheduled values indicate successful operations. All other values indicate an error condition.

See also:

  • ApiErrorService — Retrieve the error related to the last operation even if the method did not return an error code.
  • GemErrorExtension.isErrorCode — Check if an integer value is a valid error code.
