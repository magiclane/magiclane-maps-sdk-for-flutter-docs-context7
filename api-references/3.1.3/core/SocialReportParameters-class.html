SocialReportParameters

SocialReportParameters class Core

Parameters for a social-report overlay item.

Contains fields commonly provided for user-generated social reports such as the report type, optional TTS text (tts), owner information, a numeric score, optional coordinates, and lifecycle timestamps (created/updated/expiry).
